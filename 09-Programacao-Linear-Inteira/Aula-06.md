## JuMP
- Julia Mathematical Programming
- pacote de modelagem matemática para Julia

### Sintaxe do JuMP
#### ```@variable```
- define as variáveis de decisão do modelo


#### ```@constraint```
- define as restrições do modelo



#### ```@objective```
- define a função objetivo do modelo
